##求其他底数的对数

默认情况下各种语言只会提供10为底的和自然数e为底的对数api。
其他自定义底数可以使用对数的换底公式，例2为底的公式：log2(a)=log(a)/log(2)

    function logs(a,b){//以a为底，b为上面
        return Math.log(b)/Math.log(a);
    }
